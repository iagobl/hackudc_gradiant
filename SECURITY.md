# Security Policy

## Supported Versions

Only the latest version of the HackUDC Gradiant Vault is currently supported for security updates.

## Reporting a Vulnerability

We take the security of this project seriously. If you believe you have found a security vulnerability, please do NOT open a public issue. Instead, follow these steps:

1. Send an email to [INSERT SECURITY CONTACT EMAIL] with a description of the vulnerability.
2. Include details on how to reproduce the issue (exploit code, screenshots, etc.).
3. We will acknowledge receipt of your report within 48 hours.

## Disclosure Policy

We follow a coordinated disclosure policy:
- We will work with you to understand and validate the issue.
- We will provide a timeline for a fix.
- We request that you do not disclose the vulnerability publicly until a fix is released and users have had time to update.
- Once fixed, we will give credit to the reporter (unless you prefer to remain anonymous).
